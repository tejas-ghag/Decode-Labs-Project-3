# Decode Labs Project 3
### Women's E-Commerce Clothing Reviews — Data Analysis & Insights

---

## ✨ Overview

Ever wondered what customers *really think* about clothing products online?

This project analyzes real e-commerce clothing reviews to uncover patterns in customer satisfaction, ratings, recommendations, and customer feedback using Python, Data Analysis, and basic NLP techniques.

The project combines concepts learned from previous projects, including Exploratory Data Analysis (EDA), Data Visualization, and Text Analysis, while using Jupyter Notebook to document the complete analysis workflow.

---

## 📂 Dataset

Dataset Source:

https://www.kaggle.com/datasets/nicapotato/womens-ecommerce-clothing-reviews

Dataset Used:

- Womens Clothing E-Commerce Reviews.csv

The dataset contains customer reviews, ratings, recommendations, product categories, customer age information, and feedback metrics from an online clothing retailer.

---

## 📊 What This Project Does

- Cleans and prepares real-world review data
- Explores customer ratings and behavior patterns
- Identifies factors influencing product recommendations
- Visualizes customer feedback using charts and graphs
- Extracts common themes from review text using WordCloud
- Generates business insights from customer reviews

---

## 🧠 Key Insights Found

- Most customers are highly satisfied, with 4–5 star ratings dominating the dataset.
- Product recommendations strongly align with higher customer ratings.
- Some clothing departments receive significantly more customer engagement than others.
- Certain departments achieve higher customer satisfaction ratings.
- Frequently used review terms reveal common customer experiences and opinions.
